Energy Pebble API

Real-time electricity price data and color-coded indicators for smart energy consumption

How It Works

Energy Pebble fetches real-time electricity prices from Elia's grid data and converts them into simple color codes to help you optimize your energy consumption:

GREEN
YELLOW
RED
  • Green: Cheapest hours - perfect for high energy usage
  • Yellow: Average prices - moderate energy usage recommended
  • Red: Most expensive hours - avoid high energy usage

üîí Stable Color Promise

Once you see a color, it won't change for the next 8 hours. This means you can confidently plan your energy usage without worrying about colors shifting.

Smart scheduling: Use green hours for dishwashers, washing machines, electric car charging, and other energy-intensive tasks.

API Endpoints

  • GET /api/color-code - Current and upcoming color codes
  • GET /api/json - Raw electricity price data
  • GET /api/sample - Sample data for testing
  • GET /api/sample-color-code - Sample color codes
  • GET /docs - Interactive API documentation

Technical Details

Data Source: Elia Grid International (Belgian TSO)

Update Frequency: Real-time with 8-hour commitment windows

Coverage: Belgium electricity market prices

API Format: JSON responses with CORS support

Caching: Optimized for performance and stability
